Envi Smart Heater - Enhanced Home Assistant Integration

A comprehensive Home Assistant custom component for Envi Smart Heaters with enhanced API capabilities, efficient data polling, and comprehensive device monitoring.

KEY FEATURES:
• Full Control: Temperature, power, and mode control with automatic unit conversion
• Comprehensive Monitoring: 16 entities per device (1 climate + 5 binary sensors + 10 sensors)
• Performance Optimized: DataUpdateCoordinator reduces API calls by ~66%
• Professional UI: Dynamic icons, proper entity categories, organized device registry
• Advanced Features: 7 custom services, robust error handling, automatic token management

WHAT'S INCLUDED:
✅ Full climate control (temperature, state, mode)
✅ Complete device monitoring (signal strength, firmware, schedule info, WiFi, location, etc.)
✅ Efficient coordinator-based updates (single API call per cycle)
✅ Professional UI organization
✅ Extensive error handling and recovery
✅ Comprehensive documentation

API LIMITATIONS:
Some settings are read-only through the API (freeze protect, child lock, notifications, hold) and can only be changed via the Envi mobile app. These are available as binary sensors for monitoring.

REQUIREMENTS:
• Home Assistant 2023.x or later
• Envi account credentials
• aiohttp>=3.8.4

STATUS: Production Ready ✅

The integration is fully functional with all core features implemented and tested.
